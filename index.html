How to Use Your API Key

Follow these steps to get your avatar working:

// 1. Initialize the Heygen SDK with your API key
const heygenSDK = new HeygenStreamingSDK.StreamingClient({
  apiKey: 'kPXhuFy04G7' // Your API key here
});

// 2. Create a session with your avatar
const session = await heygenSDK.createSession({
  avatarName: 'Wayne_20240711'
});

// 3. Start the session with video element
await session.start({
  videoElement: document.getElementById('avatar-video')
});

Important Notes:

  • Make sure your API key has access to the Streaming API
  • The avatar name must match exactly: 'Wayne_20240711'
  • You need microphone permissions for interactive conversations
  • For production, generate access tokens server-side for security
